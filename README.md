* 这是一个maven web项目
* 简单复习一下spring+struts2+hibernate框架crud
* 纯粹xml没有任何注解
* 只有用户登录，不登陆默认为后台管理员状态(有些逻辑没注意我随意弄的)
* jsp/alipay下的文件没有使用，我只是需要index.jsp + return_url.jsp中的代码，所以放这里好复制粘贴
